== Description ==

Switips – is an international service of clever and sustainable consumption!
The module allows partners to interact with the service through the API. Allows you to transfer information about orders to the Switips service.
Before using the module, you must register
in the service https://switips.com. In the personal account of the service, you will have access to: API key, partner ID. These parameters must be specified in the module settings

The `woocommerce-switips` plugin uses the https://switips.com service for its work.
The following information is transmitted to the switips.com service:
1. Order information.
2. Calculation of agency fees.

By installing this plugin, you agree to the following documents:
Client License - https://switips.com/license
Partners' offer - https://switips.com/offer
Personal data processing policy - https://switips.com/privacy_policy
Cookie Policy - https://switips.com/cookie_policy


Switips – это международный сервис умных и осознанных покупок! После установки модуля, информация о покупках передается в сервис Switips. Перед использованием модуля, вы должны зарегистрироваться в сервисе https://switips.com и получить API key и partner ID, которые необходимо указать в настройках модуля.

Плагин `woocommerce-switips` использует для своей работы сервис https://switips.com. 
В сервис switips.com передается следующая информация:
1. Информации о заказах.
2. Калькуляция агентского вознаграждения.

Устанавливая данный плагин, вы соглашаетесь со следующими документами:
Клиентская лицензия - https://switips.com/license
Оферта партнеров - https://switips.com/offer
Политика обработки персональных данных - https://switips.com/privacy_policy
Политика в отношении файлов cookie - https://switips.com/cookie_policy

== Installation ==

1. Upload `woocommerce-switips.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Done. This plugin no requires extra configurations to work
